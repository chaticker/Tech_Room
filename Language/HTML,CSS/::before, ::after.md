### ::before
요소 앞에 내용 생성

### ::after
요소 뒤에 내용 생성

### content
요소의 전후에 ::before 나 ::after라는 의사클래스 이름을 사용하여 텍스트나 이미지 등의 콘텐츠를 삽입 할 때 사용
